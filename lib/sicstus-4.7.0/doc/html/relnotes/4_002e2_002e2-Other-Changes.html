14.18.3 Other Changes

  • SICStus Prolog is now available for 64-bit (x86_64) Windows.
  • The C API, including the foreign language interface, now uses the type SP_integer in all places where it used to use the type long int. This is a compatible change. On all platforms available in SICStus Prolog 4.2.1 the two types are identical. The types only differ on 64-bit Windows.

    As part of this change, several printf conversion specifiers are defined as C macros to allow platform independent printing of the types SP_integer, SP_uinteger and size_t. See SPRIdINTEGER et al. in the sicstus.h header.
