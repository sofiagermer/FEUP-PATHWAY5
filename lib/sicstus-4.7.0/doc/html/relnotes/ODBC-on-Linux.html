11.3 ODBC on Linux

On Linux, library(odbc) is built with unixODBC. unixODBC is an installable package on many Linux distributions, and can also be downloaded from http://www.unixodbc.org. The version differs between platform, see https://sicstus.sics.se/ for details.

You will also need to install the DBMS specific ODBC drivers.
