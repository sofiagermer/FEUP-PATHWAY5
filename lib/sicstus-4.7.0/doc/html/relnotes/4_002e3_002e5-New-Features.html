14.25.1 New Features

  • library(csv) is a new library module that provides I/O on comma-separated values (CSV) files and strings.
  • library(statistics) is a new library module that provides commonly used sample and population statistics functions.
  • CLPFD: propagation for 0/1-variables is generally faster.
