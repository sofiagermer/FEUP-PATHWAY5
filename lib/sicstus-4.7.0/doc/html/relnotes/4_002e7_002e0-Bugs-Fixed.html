14.31.3 Bugs Fixed

  • Profiling interpreted code could sometimes crash SICStus.
  • Fix critical bug while preparing to collect garbage, introduced in version 4.6.0.
  • Fix performance bug in findall/3 and friends.
  • at_end_of_line/[0,1] would sometimes fail at the end of the stream.
  • Fix issues in library(json) where it would accept invalid JSON input.
  • CLPFD:
    • - fd_closure/2 would fail on non-domain variables. Release 4.5.1 behavior restored. Some compile-time transformations were not delayed to run time as they should.
