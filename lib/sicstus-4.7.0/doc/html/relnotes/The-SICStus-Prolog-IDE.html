12 The SICStus Prolog IDE (SPIDER)

SICStus Prolog IDE, also known as SPIDER, is an Eclipse-based development environment for SICStus with many powerful features. SPIDER is meant to eventually replace the Emacs interface and the Windows spwin.exe program as the main development environment for SICStus Prolog. SPIDER was initially made available with release 4.1.0. See SICStus Prolog IDE in the SICStus Prolog Manual for more information and links.
