12.3.53 SP_load()

Synopsis

#include <sicstus/sicstus.h>

int
SP_load(char const *filename);

Calls load_files/1.

Arguments

filename

The file name, which is treated as a Prolog atom.

Return Value

See SP_query_cut_fail().
