10.20.8.8 Deprecated Jasper Predicates

jasper_new_object(+JVM,+Class,+TypeSig,+Args,-Object)   obsolescent

Creates a new Java object.

JVM

A reference to the Java VM, as obtained by jasper_initialize/[1,2].

Class

An an atom containing the fully qualified classname (i.e. package name separated with ’/’, followed by the class name), for example java/lang/String, se/sics/jasper/SICStus.

TypeSig

The type signature of the class constructor. A type signature is a string that uniquely defines a method within a class. For a definition of type signatures, see the JNI Documentation.

Args

A list of argument specifiers. See Deprecated Argument Conversions.

Object

A term on the form '$java_object'(X), where X is a Java object reference. This is the Prolog handle to the Java object. See Global vs. Local References.
