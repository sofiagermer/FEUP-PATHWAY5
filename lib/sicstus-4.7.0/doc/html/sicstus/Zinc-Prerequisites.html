10.54.1 Prerequisites

Let $MINIZINC_DIR denote the directory containing the MiniZinc distribution.

To add SICStus Prolog as a MiniZinc and FlatZinc solver, two additions to $MINIZINC_DIR are required:

  • Copy or symbolically link the SICStus specific global constraint definition directory $SP_LIBRARY_DIR/zinc/globals). The copy or symbolic link should be named $MINIZINC_DIR/share/minizinc/sicstus).
  • Copy or symbolically link the configuration file $SP_LIBRARY_DIR/zinc/sicstus.msc). The copy or symbolic link should be named $MINIZINC_DIR/share/minizinc/solvers/sicstus.msc).

Finally, make sure that $MINIZINC_DIR/bin is included in the environment variable PATH.
