11.3.23 bb_get/2

Synopsis

bb_get(:Key, -Term)

If a term is currently stored under Key, then a copy of it is unified with Term. Otherwise, bb_get/2 silently fails.

Arguments

:Key

bbkey

Term

term

Exceptions

instantiation_error

Key is not instantiated

type_error

Key is not an atom or a small integer.

See Also

ref-mdb-bbd.
