11.3.71 =:=/2   ISO

Synopsis

+Expr1 =:= +Expr2

Succeeds if the results of evaluating Expr1 and Expr2 are equal.

Arguments

Expr1

expr, must be ground

Expr2

expr, must be ground

Description

Evaluates Expr1 and Expr2 as arithmetic expressions and compares the results.

Exceptions

Arithmetic errors (see ref-ari-exc).

Examples

| ?- 1.0 + 1.0 =:= 2.

yes

| ?- "a" =:= 97.

yes
